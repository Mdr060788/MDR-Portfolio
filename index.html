Extracción y visualización de datos web

Este proyecto fué desarrollado a partir de un raspado web de la página oficial de la nación. En ella se encuentran todas las licitaciones viales del país que se publican, su estado, presupuesto, plazos y demás detalles. El raspado fue realizado utilizando Python y selenium, una vez limpios y estructurados los datos, obtuvimos las ubicaciones de cada una a través de un plugin de geolocación. Posteriormente se procedió a realizar el reporte visual en Power BI.
